# mapViewer
DT map viewer

![Imgur](https://i.imgur.com/3Pk2mki.gif)

**How to download & install :**
```
git clone https://github.com/Drigtime/mapViewer
cd mapViewer && yarn
```
**How to start it :**
```
yarn start
```
**How to build the app :**
```
yarn build-osx
yarn build-win
yarn build-linux
```
